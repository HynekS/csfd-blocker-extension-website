Why?

Let's be honest: a lot of those who post reviews or comment on this website are one of these kinds of people:

They might be offensive, boring, or just plain annoying. This extension gives you an opportunity to avoid them completely. You can make your time on this website much more enjoyable!

But seriously, How?!

Don't worry. It's really simple. First, install the extension from the Chrome Web Store.

  • 1. After the installation, you'll find three tiny dots in the top right corner of the "review" or "comment" box

  • 2. Clicking those dots will expand a menu with a single option: "Block user XY." Click on it.

  • 3. The user is now blocked. You can unblock him using the same menu that will appear if you click on the "show post" button.

    • Looks cool! What else can it do?

    To easily manage the list of blocked users, or to add users without searching for their posts, you can use the popup. The Popup can be activated by clicking the extension icon right next to the address bar.
